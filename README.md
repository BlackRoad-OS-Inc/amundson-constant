# The Amundson Constant & Framework

## Definition

```
G(n) = n^(n+1) / (n+1)^n = n / (1 + 1/n)^n

A_G = Σ G(n)/n!  for n = 0, 1, 2, ...
```

Six symbols: `n`, `1`, `+`, `/`, `^`, `()`
